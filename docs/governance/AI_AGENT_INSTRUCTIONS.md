# AI Agent Instructions — documentation-standard

## Overview

This document provides governance instructions and operational context for AI agents working in the `documentation-standard` repository.

## Core Directives

1. All changes must be made via pull request unless explicitly authorized for direct push.
2. All new markdown files must include a valid UDS frontmatter block.
3. Do not modify files outside the scope of the current phase YAML.
4. Always update `docs/planning/manifest.json` when completing a phase.
